# Digital_pathology
# Классификация гистологических патчей (digital pathology)

Репозиторий содержит решение задания по классификации фрагментов гистологических
изображений (патчей) на 9 классов:

`ADI, BACK, DEB, LYM, MUC, MUS, NORM, STR, TUM`.

Каждый патч — цветное 8-битное изображение `224×224×3`.

Основной файл решения: **`digitalpat.ipynb`**.

---

## 1. Постановка задачи

Необходимо:

1. Реализовать модель для многоклассовой классификации патчей.
2. Поддержать инференс на наборах `test`, `test_tiny`.
3. Реализовать метод `Model.load()`, который **автоматически загружает веса
   обученной модели из облачного хранилища** (Google Drive), если локальных
   файлов нет.
4. Оценить качество модели на тестовых данных.

## 2. Используемые библиотеки

В ноутбуке используются:

- `numpy`, `matplotlib`
- `torch`, `torchvision`
- `scikit-learn`
- `gdown`
- `Pillow`
- служебные: `tqdm`, `IPython.display` и др.

  В итоге реализованная модель на основе ResNet-18 и ансамбля с RandomForest
достигает balanced accuracy около **XX–YY%** на тестовом наборе. Модель устойчиво
обучается, не демонстрирует сильного переобучения и автоматически загружает
веса из облачного хранилища с помощью метода `Model.load()`, что соответствует
требованиям задания.
